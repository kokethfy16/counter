# counter
Суть максимально проста. бот считает количество сообщений, количество символов с пробелами
и без и сохраняет их в json файл.

Чтобы указать боту канал, на котором он будет считать сообщения - нужно
вставить id канала на 21 строку ( channel = client.get_channel(ваш_канал) ).


Так же бот не будет работать, если не добавить токен. В файле token.txt
указано, что с ним делать.


Чтобы авторизовать бота на сервер, нужно выполнить следующий алгоритм:
1) Создаем бота на https://discord.com/developers/applications
2) Переходим по этой ссылке https://discord.com/oauth2/authorize?client_id=YOUR_CLIENT_ID&scope=bot&permissions=8 , предварительно
изменив YOUR_CLIENT_ID на ваш id клиента, который также можно взять в настроке бота. 

И последнее:
В настройках бота на сайте переходим в раздел "bot" и ставим галочки на 'Privileged Gateway Intents' (2 галочки)
